
# 📌 newGallery

Projeto desenvolvido para praticar o uso de jQuery, especialmente para manipular links de imagens e adicioná-las dinamicamente ao layout da página.

Trata-se de uma **landing page de galeria de fotos**, com as seguintes funcionalidades:

* Inicialmente exibe 3 imagens.
* Botão **"Nova imagem"** que exibe um formulário usando efeito slideUp para inserir o link da imagem.
* Formulário com botão **Adicionar** para validar o link e adicionar a imagem ao layout, e botão **Cancelar** para esconder o formulário sem adicionar.
* Layout em linhas com 3 imagens por linha.
* Possibilidade de adicionar imagens infinitamente, porém as imagens adicionadas desaparecem após recarregar a página.

### 🔧 Tecnologias utilizadas

* HTML  
* CSS  
* JavaScript  
* jQuery

## 🛠 Funcionalidades

* ✅ Exibe 3 imagens iniciais no layout
* ✅ Formulário com efeito slideUp para adicionar nova imagem via link
* ✅ Validação do link para imagem antes de adicionar
* ✅ Layout responsivo com 3 imagens por linha
* ⏳ Persistência das imagens adicionadas não implementada (reinicia ao recarregar a página)
